# Gripper Website

This is the official website for Gripper browser extension.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- next-intl (Internationalization)
- next-themes (Dark/Light mode)
- Framer Motion (Animations)
- Lucide React (Icons)

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Features

- Responsive design
- Dark/Light mode support
- Internationalization (English & Chinese)
- Smooth animations
- SEO optimized
